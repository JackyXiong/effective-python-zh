## 翻译规约

1. 对于耳熟能详的通用术语，直接保留英文即可，不用翻译
2. 对于有争议或者容易造成误解的词汇，建议在译文旁边同时保留英文
3. 数字、英文等，与中文字符之间应该有 1 个空格
4. 注意正确使用 `的`、`地`、`得`
